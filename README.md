# wancai-admin

> wancai management system

## Build Setup

``` bash
# init project
vue init webpack wancai-admin

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

## Admin Develop
``` bash
# 安装饿了么UI组件库
npm install element-ui -S

# 安装 Vue-router路由模块
npm install vue-router -S
# 安装 Vuex 状态管理模块
npm install vuex -S
```
